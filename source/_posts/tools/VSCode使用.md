# 快捷键设置

编辑器操作:

- [ ] 隐藏左边目录栏 `command + b`
- [ ] 显示隐藏终端 <code>control + `</code>



文件操作:

- [x] 注释，使用 `command + /` 可以把选中的行都注释掉 
- [ ] 注释，使用 `shift + alt + a` 可以把选中的内容注释掉

	上面两个注释的区别就是，当选中的内容不是完整的两行的情况下，第一个会把两行都注释掉，第二个只会注释选中的内容
	
- [ ] 上下移动行 `alt + up/down`
- [ ] 复制当前行 `shift + alt + up/down`
- [ ] 删除当前行 `command + shift + k`
